# 📖 About The Project

**ACPC Navigator** is a comprehensive web application built for students participating in the **Gujarat ACPC (Admission Committee for Professional Courses)** admission process.

Navigating engineering admissions in Gujarat can be overwhelming — hundreds of colleges, dozens of branches, multiple counselling rounds, and ever-changing cutoffs.

**ACPC Navigator** brings all this information into one clean, fast, and mobile-friendly platform so students can make informed decisions with confidence.

# ✨ Features

## 🎯 College Predictor

- Predict eligible colleges based on your ACPC rank
- Branch-wise filtering
- Government and Self-Financed colleges
- Category-wise predictions
- Supports OPEN, EWS, SEBC, SC, ST and TFWS

## 📊 Cut-Off Data

- Branch-wise closing ranks
- Category-wise cutoffs
- Powerful search and filtering
- Mock Round 2026 data
- Round 3 2025 data

## 🏫 Colleges Directory

- 140+ Gujarat engineering colleges
- Fees, district and contact information
- Website, email and phone details
- Google Maps integration
- Intake information
- Placement links

## 🌿 Branch Information

- Engineering branch descriptions
- Career opportunities
- Future scope and trends
- Required skills

## 📅 Important Dates

- Complete ACPC schedule tracker
- Registration deadlines
- Merit declaration dates
- Mock round schedule
- Choice filling timelines

## 📋 Admission Guide

- Step-by-step ACPC process
- Admission workflow
- Important instructions
- Document guidance
